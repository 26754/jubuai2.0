# 剧本格式示例

本文档展示 JuBu AI 支持的标准剧本格式。

## 基本格式

### 场景头格式

```
[场景编号] [场景名称]
[地点] [时间]
```

示例：
```
01 咖啡馆
咖啡馆内，白天
```

### 对白格式

```
**角色名**：对白内容
```

### 舞台指示格式

```
（动作描述）
```

## 完整示例

```
# 第一集：相遇

## 第一场：咖啡馆

**咖啡馆内，白天**

阳光透过落地窗洒在木质地板上。

**小明**：（走进咖啡馆，环顾四周）
这里还挺安静的。

**小红**：（抬头看见小明）
小明？你怎么在这？

**小明**：（走到小红对面坐下）
我来这边办点事，没想到能遇见你。

（两人相视而笑）

## 第二场：街道

**街道外，黄昏**

夕阳西下，街道上行人稀少。

**小明**：（挥手告别）
那我先走了，改天再聊！

**小红**：（目送小明离开）
好的，路上小心。
```

## 多幕结构

```
# 第一幕：开端

## 第一场：校园

**大学校园，白天**

（学生们在草地上聊天）

...

# 第二幕：发展

## 第五场：办公室

**写字楼内，晚上**

...
```

## 注意事项

1. 使用 `#` 表示集/幕标题
2. 使用 `##` 表示场景标题
3. 场景头格式：`[地点] [时间]`
4. 对白格式：`**角色名**：（动作）对白内容`
5. 舞台指示使用 `（动作）` 或全角括号

## 支持的导入格式

- Markdown（推荐）
- Fountain（专业剧本格式）
- JSON（结构化数据）
